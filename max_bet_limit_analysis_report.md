## 1. 问题背景

部分商家会将弹珠机的最大投珠数限制为低于99的某个值，常见限制有：
- **50珠**、**40珠**、**30珠**、**20珠**、**15珠**、**10珠** 等

这一限制直接影响：
- 积分卡策略的最优投注量是否能达到
- 珠子策略在正EV场景下的收益放大效应
- 玩家的整体收益水平

## 2. 理论分析：积分效率损失

### 核心原理

积分卡公式: `cards = min(floor(multiplier × bet / T), J)`

由于 `floor()` 的阶梯效应，最优投注量并不总是越大越好，而是对齐到
恰好跨过下一个积分卡阶梯的值。当 `max_bet` 被限制时：
- 若限制值 ≥ 各倍数的最优投注 → **无影响**
- 若限制值 < 某些倍数的最优投注 → **效率下降**

> **关键结论**: `max_bet ≥ 10` 时积分卡策略效率完全不受影响。商家限制在 10 以上等于没有限制。

### 各倍数的积分卡阶梯

每个阶梯显示：获得 N 张积分卡所需的最低投注量

**2x**: `10珠=1卡` → `20珠=2卡` → `30珠=3卡` → `40珠=4卡` → `50珠=5卡` → `60珠=6卡` → `70珠=7卡` → `80珠=8卡` → `90珠=9卡`
**4x**: `5珠=1卡` → `10珠=2卡` → `15珠=3卡` → `20珠=4卡` → `25珠=5卡` → `30珠=6卡` → `35珠=7卡` → `40珠=8卡` → `45珠=9卡` → `50珠=10卡`
**6x**: `5珠=1卡` → `7珠=2卡` → `10珠=3卡` → `14珠=4卡` → `17珠=5卡` → `20珠=6卡` → `24珠=7卡` → `27珠=8卡` → `30珠=9卡` → `34珠=10卡`
**8x**: `5珠=2卡` → `5珠=2卡` → `8珠=3卡` → `10珠=4卡` → `13珠=5卡` → `15珠=6卡` → `18珠=7卡` → `20珠=8卡` → `23珠=9卡` → `25珠=10卡`
**10x**: `5珠=2卡` → `5珠=2卡` → `6珠=3卡` → `8珠=4卡` → `10珠=5卡` → `12珠=6卡` → `14珠=7卡` → `16珠=8卡` → `18珠=9卡` → `20珠=10卡`

## 6. 核心结论与应对建议

### 6.1 商家限制的实际影响分级

| 限制等级 | max_bet范围 | 积分卡策略影响 | 珠子策略影响 | 严重程度 |
|---------|-----------|-------------|-----------|---------|
| 无感限制 | ≥10 | 无损失 | 正EV场景略受限 | ⭐ 无影响 |
| 轻度限制 | 30~9 | 0~5%损失 | 正EV收益受限 | ⭐⭐ 轻微 |
| 中度限制 | 15~29 | 5~20%损失 | 正EV严重受限 | ⭐⭐⭐ 中等 |
| 严重限制 | 10~14 | 20~40%损失 | 正EV几乎无法利用 | ⭐⭐⭐⭐ 严重 |
| 极端限制 | 5~9 | 40%+损失 | 策略完全失效 | ⭐⭐⭐⭐⭐ 极端 |

### 6.2 玩家应对策略

| 场景 | 建议 |
|------|------|
| max_bet ≥ 30 | 正常使用积分卡策略，影响极小 |
| max_bet = 15~29 | 积分卡策略仍有效，注意选择对齐阶梯的投注量 |
| max_bet = 10~14 | 积分卡策略效率明显下降，考虑是否值得继续玩 |
| max_bet < 10 | 策略价值极低，不建议在此类机器上消费 |
| 偏斜分布+限制 | 珠子策略受损更大（无法放大正EV），请降低期望 |

### 6.3 商家视角

- 限制 max_bet 是商家**最温和**的抽水手段之一（相比改倍率概率或减灯格数）
- 对积分卡策略：限制到 10~15 会显著削弱玩家效率
- 对珠子策略：限制 max_bet 可以有效阻止玩家在正EV场景下翻盘
- 商家设限越低，玩家的策略空间越小，长期亏损越确定
