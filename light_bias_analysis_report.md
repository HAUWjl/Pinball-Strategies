# 灯位偏置分析报告：商家改变某些灯出现概率产生的影响

> Light Position Bias Analysis for 12-Slot Pinball Machine

## 研究概述

在12口弹珠机中，每次按下按钮确定倍率后，机器随机选择若干格位亮灯。本报告研究一种**极其隐蔽的操控手段**——**灯位偏置**：商家通过修改程序/电路板，让灯更倾向于亮在弹珠不容易落入的格位上，从而在不改变亮灯数量和倍率概率的前提下，系统性降低玩家的中奖率。

### 为什么这种操控最隐蔽？

| 检查项 | 是否通过 |
|--------|----------|
| 亮灯数量正确（2x=4灯, 4x=3灯...） | ✅ 通过 |
| 倍率出现频率正常 | ✅ 通过 |
| 每个灯位看起来都会亮 | ✅ 通过 |
| 实际中奖率正常 | ❌ 偏低（但可归因于"运气不好"） |

## 第一部分：操控原理

### 操控方式

1. 商家统计弹珠的物理落点分布（长期运行数据）
2. 修改灯位选择算法，使**高概率落点的格位更少被选为灯位**
3. 保持亮灯数量不变（2x仍亮4格、4x仍亮3格...）

### 数学模型

定义：
- $p_i$ = 弹珠落入格位 $i$ 的物理概率 ($i = 0..11$)
- $w_i$ = 格位 $i$ 被选为灯位的权重

**公平选灯**：$w_i = 1/12$ 对所有 $i$

$$P(\text{中奖} \mid n\text{灯}) = \frac{n}{12} \quad \text{（与落点分布无关）}$$

**偏置选灯**：$w_i \propto (1 - \text{bias} \times p_i / \max(p))$

$$P(\text{中奖} \mid n\text{灯}, \text{偏置}) < \frac{n}{12}$$

> **关键结论**：公平选灯下，无论弹珠落点分布如何偏斜，中奖概率恒等于 $n/12$。但偏置选灯打破了这个等式，且落点越偏斜，偏置效果越显著。

## 第七部分：灯位偏置检测方法

### 方法一：灯位频率卡方检验

记录每个格位亮灯的次数，与理论期望值比较。

- 100局中平均亮灯总次数约 **296次**
- 每格期望次数 = **24.7次**
- $\chi^2$ 统计量 = $\sum \frac{(\text{观测值} - \text{期望值})^2}{\text{期望值}}$
- 自由度 = 11，α=0.05时临界值 = **19.68**
- 若 $\chi^2 > 19.68$，有95%把握认为灯位选择不公平

### 方法二：落点-灯位相关性分析

1. 记录每局亮灯格位和弹珠落点
2. 计算每个格位的亮灯频率 vs 落点频率
3. 若存在显著负相关（$r < -0.5$ 且 $p < 0.05$），确认偏置
4. 需200+局数据

### 2. 关键特性

- **最隐蔽的操控方式**：灯数正确、倍率正常，视觉和简单统计均无法发现
- **落点越偏斜，偏置效果越强**：两者相互放大
- **均匀落点下偏置无效**：所有格位落球概率相等时无法利用
- **老旧/磨损机器更容易被操控**：物理偏斜更大

### 3. 操控方式对比

| 操控方式 | 影响程度 | 隐蔽性 | 检测难度 |
|:--------:|:--------:|:------:|:--------:|
| 倍率概率调整 | 高 | 中 | 50局可检测 |
| 亮灯数量减少 | 很高 | 低 | 一眼可见 |
| **灯位偏置** | **中~高** | **很高** | **100~200局** |
| 组合操控 | 极高 | 取决于组合 | 需多维度分析 |

### 4. 玩家实操建议

1. **前20局用最小投注(5珠)试探**
2. **记录每局亮灯格位编号**（手机拍照最简单）
3. 每20局检查各格位亮灯次数是否均匀
4. 如果某些格位明显很少亮灯，且这些格位恰好是你中奖最多的格位 → **高度怀疑灯位偏置操控**
5. 如果综合中奖率低于理论值20%以上，**建议更换机器**
6. **使用V2策略**，即使存在偏置也能自动减少损失
